Conectar con Google Sheets

Para ver tus datos reales necesitas el ID de tu documento y una API Key de Google Sheets. Si no los tienes aún, puedes explorar el dashboard con datos de demo.

Cómo obtener la API Key:
1. Ve a console.cloud.google.com
2. Crea un proyecto → habilita "Google Sheets API"
3. Credenciales → Crear credencial → Clave de API
4. Tu documento debe estar compartido como "Cualquier persona con el link puede ver"
